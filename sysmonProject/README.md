# Procfs/System Monitor

Простой мониторинг системы на основе `/proc` filesystem.

## Описание

**sysmon** — это утилита для мониторинга ресурсов Linux системы путем чтения информации из `/proc`:
- CPU статистика (user, system, iowait, idle)
- Память (total, free, available, used %)
- Список активных процессов
- Системная информация и время

## Особенности

- ✅ Чтение CPU статистики из `/proc/stat`
- ✅ Чтение информации о памяти из `/proc/meminfo`
- ✅ Перечисление активных процессов
- ✅ Красивый форматированный вывод
- ✅ Минимальные зависимости (только glibc)

## Компиляция

```bash
gcc -o sysmon sysmon.c -Wall
```

Или используя Makefile:
```bash
make
```

## Использование

```bash
./sysmon
```

Программа выведет:
- Текущее время
- Статистику по CPU (%)
- Статистику по памяти (KB и %)
- Список первых 10 запущенных процессов
